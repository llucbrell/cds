Charlie Don't Surf

CDS is an advanced tool for managing, building prompts, and testing language models. Aimed at developers, it optimizes and evaluates the efficiency of language models, assisting in the management of tests and results.

Features

Test Administration

Manage collections and tests, allowing the creation, editing, and deletion of collections. Configure detailed tests with specific parameters and customized conditions.

Prompt Construction

Use Jinja templates to create highly flexible custom prompts. Insert variables, set conditions, and reuse components for dynamic and adaptable construction.

Load data directly onto Jinja templates using CSV files.

Language Model Testing

Run automated tests to evaluate the functionality and performance of models under various conditions. Launch test suites against APIs of different models for comprehensive evaluation.

The results are stored and presented in an organized manner, providing detailed reports to identify areas for improvement and optimize the models.

Technology Used

The application is based on a robust infrastructure with Flask for managing and executing tests, SQLAlchemy for the database, and Jinja for building prompts. On the client side, JavaScript and Moment.js handle date and time conversion, while Bootstrap ensures an intuitive and responsive interface.

It is a solution for organizations looking to maximize the performance and reliability of their language models, ensuring precise evaluation and continuous optimization.
